About Me!

Saya adalah Programmer lulusan Manajemen Informatika dari Politeknik Astra dengan pengalaman praktis dalam pengembangan aplikasi web, desktop, dan mobile. Saya memiliki keahlian dalam membangun aplikasi web menggunakan React, Next.js, Laravel, dan Spring Boot, serta terbiasa mengintegrasikan sistem dengan database dan REST API.

Saya pernah terlibat dalam pengembangan Sistem Manajemen Turnamen Golf berbasis web yang berhasil mendigitalisasi proses manual menjadi sistem real-time dan meningkatkan efisiensi hingga 80%. Selain itu, saya berpengalaman menggunakan Docker dan Azure DevOps untuk mendukung proses CI/CD dan deployment aplikasi.

Saya terbiasa bekerja dalam tim, memiliki kemampuan problem solving yang baik, serta mampu beradaptasi dengan teknologi baru. Dengan latar belakang akademik dan proyek nyata, saya berkomitmen untuk membangun aplikasi web yang efisien, scalable, dan user-friendly.
